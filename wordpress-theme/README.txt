== Descrição ==

Tema WordPress personalizado para páginas de vendas VSL (Video Sales Letter) do Rhuan Christo.

Este tema foi desenvolvido especialmente para criar landing pages de alta conversão com foco em vendas através de vídeos e copywriting persuasivo.

== Instalação ==

1. Faça upload da pasta 'rhuan-vsl' para o diretório '/wp-content/themes/'
2. Ative o tema através do menu 'Aparência' no WordPress
3. Configure o tema em 'Aparência > Personalizar'

== Recursos ==

* Design responsivo e mobile-first
* Seções personalizadas para landing pages
* Campos customizados para hero e CTA
* Otimizado para conversão
* Compatível com page builders
* Animações suaves
* SEO friendly

== Template de Landing Page ==

Para usar o template de landing page:
1. Crie uma nova página
2. No editor, selecione o template "Landing Page"
3. Preencha os campos personalizados nas seções Hero e CTA
4. Adicione seu conteúdo no editor
5. Publique!

== Personalização ==

Você pode personalizar cores, fontes e estilos editando o arquivo style.css.

Para adicionar campos customizados adicionais, edite o arquivo functions.php.
